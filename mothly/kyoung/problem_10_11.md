# 문제 10. [서술형] DRF CRUD에서 사용하는 HTTP method 종류 4가지와 각각 어떤 목적으로 사용하는지 작성하시오.
1. GET: 정보를 조회할 때 사용한다.
2. POST: 새로운 정보를 등록할 때 사용한다. 
3. PUT: 정보를 수정할 때 사용한다.
4. DELETE: 정보를 삭제할 때 사용한다.
